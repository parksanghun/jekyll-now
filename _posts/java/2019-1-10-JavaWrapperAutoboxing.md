#### Wrapper 클래스란?
- 원래 자료형에다가 랩을 씌운 클래스라고 하여 Wrapper 클래스인 것이다

| 기본형 타입    | wrapper class |
|-------------|---------------|
| byte        | Byte          |
| short       | Short         |
| int         | Integer       |
| long        | Long          |
| float       | Float         |
| double      | Double        |
| char        | Charater      |
| boolean     | Boolean       |
| void        | Void          |
